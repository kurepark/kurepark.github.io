펀딩 등록 페이지

사용자가 직접 펀딩을 등록할 수 있는 서비스 입니다.

펀딩에 필요한 정보들을 입력하는 서비스이다 보니 다양한 input type을 사용해 볼 수 있었고 사용자들의 입력 여부를 체크하여 버튼이 활성화 되도록 해야했기 때문에 입력값을 체크하는 스크립트를 만들어 사용하였습니다.

세 종류의 펀딩 페이지를 시간차를 두고 작업하였는데 이 또한 기업, 투자자 홈과 마찬가지로 처음 css, js 작성 시 별도의 파일로 작성하였다가 증권형 펀딩 개설 프로젝트 진행 시에 css, js 파일 통합작업을 다시 진행하였습니다
